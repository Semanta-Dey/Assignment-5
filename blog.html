Question-1: What are the different ways to select an element in the DOM?

In DOM there are different ways to select an element . They are:
1."document.getElementById("ID")",
2."document.getElementsByClassName("class name")",
3."document.getElementsByName()",
4."document.getElementsByTagName()",
5."document.getElementsByTagNameNS()",
6."document.querySelectorAll()",
7."document.querySelector()".

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.By using innerHTML we get and set inner HTML of an element.
2.By using innerText we get and set just inner text of an element.
3.By using textContent we get the text content of an element and also get the hidden content.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript that allows you to handle events at a higher level instead of adding event listeners to multiple child elements. This is useful when dealing with dynamically created elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in JavaScript where an event starts at the target element and then propagates through its ancestors until it reaches the root.
